# 基于FPGA和STM32的信号分离系统

>&emsp;&emsp;这个是2023年电赛H题的源代码，我们一开始是打算用STM32做FFT+DDS的方案，但是最后发现由于DDS的频率精度不够，导致无法完成同频稳定显示的要求，后面在其他队伍的启发下用FPGA的DAC做输出，即可达到高精度频率的输出。  
>&emsp;&emsp;由于我们先完成了STM32的FFT部分，后面才改为FPGA，由于时间有限，我们就用STM32做FFT运算，将求得的频率、波形信息通过串口传输给FPGA，这样就省了FPGA做FFT的部分（其实当时也不会hh）。这样，在最终测评的时候用普联的示波器就基本看不到频飘了。

一、标题写法：
第一种方法：
1、在文本下面加上 等于号 = ，那么上方的文本就变成了大标题。等于号的个数无限制，但一定要大于0个哦。。
2、在文本下面加上 下划线 - ，那么上方的文本就变成了中标题，同样的 下划线个数无限制。
3、要想输入=号，上面有文本而不让其转化为大标题，则需要在两者之间加一个空行。
另一种方法：（推荐这种方法；注意⚠️中间需要有一个空格）
关于标题还有等级表示法，分为六个等级，显示的文本大小依次减小。不同等级之间是以井号  #  的个数来标识的。一级标题有一个 #，二级标题有两个# ，以此类推。
例如：
# 一级标题  
## 二级标题  
### 三级标题  
#### 四级标题  
##### 五级标题  
###### 六级标题

7、换行（建议直接在前一行后面补两个空格）
直接回车不能换行，  
可以在上一行文本后面补两个空格，  
这样下一行的文本就换行了。
或者就是在两行文本直接加一个空行。
也能实现换行效果，不过这个行间距有点大。
